**Activités interactives de compréhension orale pour apprenants francophones**

## EN — What's in this repo

Self-contained HTML activities for A2–B2 learners. No login, no installation — each file opens directly in a browser and provides instant feedback. All activities can be downloaded and deployed on any LMS (Moodle, etc.) as a file resource.

| Activity | Topic | Level |
|---|---|---|
| [The Big Bang Theory — S01E01](https://fxpriour.github.io/listening-activities/tbbt.html) | American sitcom, pilot episode (first 4 min.) | A2–B1 |
| [The Byronic Hero](https://fxpriour.github.io/listening-activities/byronic_hero.html) | Literature & cultural history (YouTube, 0:00–6:08) | A2–B2 |
| [Bram Stoker's *Dracula*](https://fxpriour.github.io/listening-activities/dracula.html) | Gothic literature, 19th century (TED-Ed, up to 3:21) | A2–B2 |
| [Mary Shelley's *Frankenstein*](https://fxpriour.github.io/listening-activities/frankenstein.html) | Gothic literature, Romantic period (TED-Ed, up to 2:55) | A2–B2 |
| [Lovecraft: *Titan of Terror*](https://fxpriour.github.io/listening-activities/lovecraft.html) | Cosmic horror, Cthulhu Mythos (TED-Ed) | A2–B2 |

**Exercise types:** fill in the blanks · multiple choice · open questions  
**Languages:** instructions in English; some questions allow — or require — answers in French

> **Note on open questions:** these pages show a suggested answer for self-correction. A fully interactive version — with AI-powered evaluation of written responses — is available on Moodle for enrolled students. The GitHub version omits this feature because embedding an API key in a public file would expose it to misuse.

## FR — Contenu du dépôt

Activités HTML autonomes pour apprenants de niveau A2–B2. Aucune installation requise — chaque fichier s'ouvre dans un navigateur et propose une correction instantanée. Les fichiers peuvent être téléchargés et déployés sur tout LMS (Moodle, etc.) comme ressource de cours.

| Activité | Thème | Niveau |
|---|---|---|
| [The Big Bang Theory — S01E01](https://fxpriour.github.io/listening-activities/tbbt.html) | Sitcom américaine, épisode pilote (4 premières min.) | A2–B1 |
| [The Byronic Hero](https://fxpriour.github.io/listening-activities/byronic_hero.html) | Littérature et histoire culturelle (YouTube, 0:00–6:08) | A2–B2 |
| [Bram Stoker's *Dracula*](https://fxpriour.github.io/listening-activities/dracula.html) | Littérature gothique, XIXe siècle (TED-Ed, jusqu'à 3:21) | A2–B2 |
| [Mary Shelley's *Frankenstein*](https://fxpriour.github.io/listening-activities/frankenstein.html) | Littérature gothique, Romantisme (TED-Ed, jusqu'à 2:55) | A2–B2 |
| [Lovecraft: *Titan of Terror*](https://fxpriour.github.io/listening-activities/lovecraft.html) | Horreur cosmique, Mythe de Cthulhu (TED-Ed) | A2–B2 |

**Types d'exercices :** texte à trous · QCM · questions ouvertes  
**Langues :** consignes en anglais ; certaines questions acceptent — ou exigent — une réponse en français

> **Note sur les questions ouvertes :** une réponse suggérée est fournie pour l'autocorrection. La version Moodle intègre une évaluation automatisée par IA — cette fonctionnalité est absente ici pour des raisons de sécurité (clé API publiquement exposée).
