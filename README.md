## EN — What's in this repo

Self-contained HTML activities for A2–B1 learners. No login, no installation — each file opens directly in a browser and provides instant feedback. All activities can be downloaded and deployed on any LMS (Moodle, etc.) as a file resource.

| Activity | Topic | Level |
|---|---|---|
| [The Big Bang Theory — S01E01](https://fxpriour.github.io/listening-activities/tbbt.html) | American sitcom, pilot episode (first 4 min.) | A2–B1 |
| [The Byronic Hero](https://fxpriour.github.io/listening-activities/byronic_hero.html) | Literature & cultural history (YouTube, 0:00–6:08) | A2–B1 |

*More activities added regularly.*

**Exercise types used:** fill in the blanks · multiple choice · open questions · comparison tables  
**Languages:** instructions in English; some questions allow answers in French

## FR — Contenu du dépôt

Activités HTML autonomes pour apprenants de niveau A2–B1. Aucune installation requise — chaque fichier s'ouvre dans un navigateur et propose une correction instantanée. Les fichiers peuvent être téléchargés et déployés sur tout LMS (Moodle, etc.) comme ressource de cours.

| Activité | Thème | Niveau |
|---|---|---|
| [The Big Bang Theory — S01E01](https://fxpriour.github.io/listening-activities/tbbt.html) | Sitcom américaine, épisode pilote (4 premières min.) | A2–B1 |
| [The Byronic Hero](https://fxpriour.github.io/listening-activities/byronic_hero.html) | Littérature et histoire culturelle (YouTube, 0:00–6:08) | A2–B1 |

*Nouvelles activités ajoutées régulièrement.*

**Types d'exercices :** texte à trous · QCM · questions ouvertes · tableaux comparatifs  
**Langues :** consignes en anglais ; certaines questions acceptent une réponse en français
